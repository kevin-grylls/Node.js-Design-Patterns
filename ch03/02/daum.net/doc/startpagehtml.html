인터넷 익스플로러

인터넷 익스플로러 환경에서 다음을 시작페이지로 설정하는 방법은 아래와 같습니다.

  1. 1. 인터넷 익스플로러를 띄웁니다.
  2. 2. 메뉴에서 ‘도구>인터넷옵션’을 클릭합니다.
  3. 3. [일반] 탭의 홈페이지 URL 입력란에 ‘https://www.daum.net’을 입력 후 확인 버튼을 클릭합니다.
  4. 4. 인터넷 익스플로러를 새로 띄워보면, 다음으로 시작페이지가 변경된 것을 확인하실 수 있습니다.

엣지 브라우저

엣지 브라우저 환경에서 다음을 시작페이지로 설정하는 방법은 아래와 같습니다.

  1. 1. 엣지 브라우저를 띄웁니다.
  2. 2. 우측 상단의 [아이콘] 클릭후 하단의 [설정]을 클릭합니다.
  3. 3. 다른 프로그램으로 열기 항목의 [특정 페이지>사용자 지정]으로 수정합니다.
  4. 4. URL 입력 영역에 'https://www.daum.net'을 입력합니다.
  5. 5. 엣지 브라우저를 새로 띄워보면, 다음으로 시작페이지가 변경된 것을 확인하실 수 있습니다.

구글 크롬

구글 크롬 환경에서 다음을 시작페이지로 설정하는 방법은 아래와 같습니다.

  1. 1. 크롬 브라우저를 띄웁니다.
  2. 2. 우측 상단의 [아이콘>설정]을 클릭합니다.
  3. 3. 설정 페이지에서 [시작 그룹>특정 페이지 또는 페이지 모음 열기>새 페이지 추가]를 클릭합니다.
  4. 4. 팝업 창에서 'https://www.daum.net'을 입력하고 추가 버튼을 클릭합니다.
  5. 5. 크롬 브라우저를 새로 띄워보면, 다음으로 시작페이지가 변경된 것을 확인하실 수 있습니다.

사파리 브라우저

사파리 브라우저 환경에서 다음을 시작페이지로 설정하는 방법은 아래와 같습니다.

  4. 4. 사파리 브라우저를 새로 띄워보면, 다음으로 시작페이지가 변경된 것을 확인하실 수 있습니다.

오페라 브라우저

오페라 브라우저 환경에서 다음을 시작페이지로 설정하는 방법은 아래와 같습니다.

  1. 1. 오페라 브라우저를 띄웁니다.
  2. 2. 우측 상단의 아이콘 [시작페이지 사용자 설정] 메뉴를 클릭합니다.
  3. 3. 우측 하단의 아이콘 [브라우저 설정으로 이동]을 클릭합니다.
  4. 4. [시작 시>특정 페이지 또는 페이지 세트 열기-페이지 설정]을 클릭합니다.
  5. 5. 팝업창에 새 페이지 추가 URL에 ‘www.daum.net’을 입력하고 확인 버튼을 누릅니다.
  6. 6. 오페라 브라우저를 새로 띄워보면, 다음으로 시작페이지가 변경된 것을 확인하실 수 있습니다.
